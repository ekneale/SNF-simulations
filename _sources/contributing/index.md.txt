# Contributing and Development

This section of the documentation provides information for developers who want to contribute to the SNF simulations project.

Contributions are always welcome. All development of the package takes place on GitHub ([https://github.com/ekneale/SNF-simulations](https://github.com/ekneale/SNF-simulations)), where you can find the source code, report issues, and submit pull requests.
